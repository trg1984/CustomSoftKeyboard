Custom Soft Keyboard Technical Demo

This demo creates a custom soft keyboard on touch devices. It uses Modernizer to detect the touch capability, and bypasses the soft keyboard provided by the OS by unfocusing the input fields on focus.

The keyboard is disabled on non-touch devices by default. It can be forced on by setting config.alwaysOn to true. It has been developed and tested on Android 5.0 (Lollipop) and 6.0 (Marshmallow).

Custom layouts can be made by creating a HTML snippet for the keyboard and defining its path in config.layoutPath. See numpad.html.snippet for an example.

As this is a technical demo, it is not finished, polished or complete. See issues for a list of bugs.
